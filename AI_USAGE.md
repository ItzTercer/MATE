# Uso de Inteligencia Artificial - Proyecto MATE

El uso de herramientas de IA en este proyecto (Hito 1):

1. **Formateo de documentación:**
   Se utilizó para definir y aplicar el formato de bloques de código tipo `bash` en el archivo `README.md`. Esto permite que las instrucciones de instalación y ejecución sean visualmente claras y fáciles de seguir para el usuario final.

2. **Lógica para Raíces con Radicales Negativos:**
   Asistencia en la implementación de la función `raiz_n`. La IA ayudó a estructurar la lógica necesaria para manejar radicales negativos cuando el índice es impar, superando la limitación de la función nativa `pow()` mediante la extracción del signo y el cálculo sobre el valor absoluto.

3. **Procesamiento de strings:**
   Sugerencia técnica para el uso de la función `atof` dentro de `simbolos_unicos()`. La herramienta permitió implementar la conversión de cadenas de caracteres a valores de tipo `double`, facilitando la interpretación de constantes literales (`e`, `pi`) y números ingresados por el usuario.
